Description of HTML tags

1. title
This tag adds a title for the page. It's what text shows in the tab.
2. h1
This tag makes the text inside of it larger then any other text on the screen.
3. ul
This tag adds an unordered list. It can be continously used to make the list longer
4. abbr
This tag is used to abbreviations of works. You can use it to add a note to help explain the word.
5. cite
This tag is used to cite a source. It can be used to source who a quote is from!
